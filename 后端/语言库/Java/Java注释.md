
| 注释类型            | 区别                                    |
| --------------- | ------------------------------------- |
| 单行：//           | 只需要注释一行时使用                            |
| 多行注释：/*  * /    | 需要注释多行时使用                             |
| 文档注释：/**    * / | 一般用在类，方法、成员变量上，且在其上的注释内容会被提取到程序说明文档中去 |
