
1. **默认构造函数的存在条件与规则**：
    - 无显式定义 -> 有默认无参构造。
    - 有显式定义 -> 无默认无参构造。
2. **构造函数的语法定义**：
    - 必须与类同名。
    - **对不能有返回类型**（包括`void`）。
3. **继承体系下的构造函数调用规则**：
    - **子类不继承父类构造函数**。
    - 子类构造函数必须调用父类构造函数（通过`super`），且必须是第一行语句。
    - 如果父类没有无参构造，子类必须显式调用父类的有参构造。
4. **方法重写（Override）的概念**：
    - 重写仅适用于实例方法，不适用于构造函数、静态方法、私有方法等。
