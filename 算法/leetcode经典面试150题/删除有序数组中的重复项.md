依旧是使用快慢指针
由于第一个位置的数字一定是放在数组内的，所以快慢指针都从数组位置1开始
快指针将第i个元素与第i-1个元素判断是否相同，相同的话就不执行操作，不相同就给慢指针位置赋值

```cpp
class Solution {
public:
    int removeDuplicates(vector<int>& nums) {
        int slow = 1;
        for(int i = 1; i < nums.size(); i++){
            if(nums[i] != nums[i - 1]){
                nums[slow] = nums[i];
                slow++;
            }
        }
        return slow;
    }
};
```
