#轮转数组
给定一个整数数组 `nums`，将数组中的元素向右轮转 `k` 个位置，其中 `k` 是非负数。

注意处理超出数组总数的情况

解法：三次反转法
![[三次反转法.png]]

```cpp
class Solution {
public:
    void rotate(vector<int>& nums, int k) {
        int sum = nums.size();
        k = k % sum;
        if(k == 0) return;
        reverse(nums, 0, sum-1);
        reverse(nums, 0, k-1);
        reverse(nums, k, sum-1);
    }

    void reverse(vector<int>& nums, int start, int end){
        while(start < end){
            swap(nums[start], nums[end]);
            start++;
            end--;
        }
    }
};
```
